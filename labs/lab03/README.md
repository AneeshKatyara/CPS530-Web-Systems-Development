## Lab 03 – Responsive Web Design

This lab focuses on building a responsive website using both pure HTML/CSS and the Bootstrap framework.
The project implements a “Pet Adoption Services” website that adapts to desktop, tablet, and mobile screen sizes.

Part A demonstrates responsiveness using CSS Flexbox and media queries, while Part B uses Bootstrap’s grid system and components such as a carousel.

**Live Pages:**
- Part A (HTML & CSS): https://www.cs.torontomu.ca/~akatyara/lab03/lab03a.html
- Part B (Bootstrap): https://www.cs.torontomu.ca/~akatyara/lab03/lab03b.html

**Technologies:** HTML, CSS, Bootstrap
